## 🧭 策略思考：如何更贴近投资？

### 1. 化繁为简：策略的本质

策略研究的底层逻辑，归根结底是 **DDM（股利贴现模型）**。

> [!NOTE]
> **核心定理：策略即"预期差"**  
> 大势研判的本质，在于**识别特定时段的主导变量**，并精准把握市场对该变量的 **"预期差"**。

---

## 🧩 DDM 三要素深度拆解

| 要素 | 核心驱动 | 预期差来源 |
| :---: | :--- | :--- |
| 🏭 **分子端：企业盈利** | 经济周期、宏观政策（财政/产业） | 市场对业绩增速拐点的判断误差 |
| 💰 **分母端①：流动性** | 广义流动性（社融/M2）+ 狭义流动性（银行间/利率） | 资金供需关系的边际变化 |
| 🎢 **分母端②：风险偏好** | 量化指标体系 + 定性因素（地缘/政策） | 股权风险溢价（ERP）的极端偏离 |

### 🎯 大势研判的本质 = 回答两个问题

> [!IMPORTANT]
> 市场在未来一段时间，能否赚到 **分子端（业绩增长）** 的钱？能否赚到 **分母端（估值扩张）** 的钱？

当前的市场核心矛盾是什么？预期差在哪里？  
**答案 = 对市场预期定价的深度理解。**

## 📊 经济基本面：需求视角的分析框架

中国经济经典分析框架，以 **需求侧三驾马车** 为起点：

| 引擎 | 关键特征 | 主要影响因素 |
| :---: | :--- | :--- |
| 🚢 **出口** | 中国经济传统双引擎之一 | 全球经济景气度；2018年以来受中美贸易摩擦扰动 |
| 🏗️ **投资** | 中国经济传统双引擎之一 | **三大支柱**：制造业投资 · 房地产投资 · 基建投资 |
| 🛒 **消费** | 被动力量逐渐上升 | 就业状况 · 可支配收入 · 未来收入预期 |

> 🔗 **关键链条**：出口与制造业投资高度关联，形成 **"出口 → 制造业"** 的核心传导链条。出口和投资的波动相对较大，是影响经济基本面的核心变量。

### 核心理念

> A股盈利周期的位置与方向，是 A股大势研判最重要的**锚**。

### 📆 中国经济增长的周期嵌套

| 周期类型 | 时间跨度 | 核心驱动 |
| :---: | :---: | :--- |
| **短周期** | 约 3-4 年 | 库存周期（基钦周期） |
| **中周期** | 约 7-11 年 | 信用/债务周期 · 产能周期（朱格拉周期） |
| **长周期** | 约 15-25 年 | 库兹涅茨周期（房地产/人口） |

## 📜 A股盈利历史回测（2008-2025）

> 下表展示了历次盈利底部与市场底部的关系：**市场的底部很少晚于盈利的底部出现**，因此盈利底部是底部框架的核心构成。

| 年份 | 全年盈利增速 | 关键节点与市场表现 |
| :---: | :---: | :--- |
| **2008** | ≈ -35% | 7月政治局会议转向 **"一防一控"**；9月年内首次降准；11月国常会宣布 **"四万亿"** 计划 |
| **2011** | ≈ +2% | 11月年内首次降准；12月政治局会议定调 **"增长优先"** |
| **2018** | ≈ -5% | 4月年内首次全面降准；7月政治局会议定调 **"六稳"**；12月强调财政 **"加力提效"** |
| **2022** | ≈ -15% 至 -29% | 主要指数全线下跌，呈 **"W"形** 走势。**盈利下行至负增长，赚业绩钱极难** |
| **2023** | ≈ -3.70% | 延续调整，**北证50** 一枝独秀 |
| **2024** | ≈ +12.67% | **"9·24"政策刺激**，市场由跌转涨 |
| **2025** | ≈ +18.41% | **"科技牛"** 行情，创业板指大涨近 50% |

> [!WARNING]
> **2022年的启示**：盈利下行是确定性方向，赚业绩增长的钱几乎不可能。此时大势研判的重心，需从分子端转向分母端——评估估值是否具备扩张条件。

## 🔬 盈利分析：从微观财报读懂三个中周期

### 结构视角：三个中周期的相互影响与牵制

| 周期 | 核心定义 | 关键观测指标 |
| :---: | :--- | :--- |
| 🏗️ **产能周期** | 与固定资产投资增速最相关的中周期 | 构建资产现金流 → 在建工程 → 固定资产（三步走） |
| 📦 **库存周期** | 企业主动/被动补库与去库的波动 | 产成品库存 · 工业企业利润 |
| 💳 **偿债周期** | 企业杠杆率与债务偿还压力 | 资产负债率 · 利息保障倍数 · 现金流覆盖 |

### 🔄 三周期联动机制

> 三个中周期并非孤立运行，而是**相互影响、相互牵制**，共同决定了盈利的中期线索：

```mermaid
flowchart LR
    A[产能周期<br>（供给端扩张）] --> B[库存周期<br>（供需平衡）]
    B --> C[偿债周期<br>（财务约束）]
    C -->|制约投资意愿| A
    A -->|产能释放影响| B
    B -->|盈利影响偿债| C

💡 简而言之：产能决定供给上限，库存反映当期供需错配，偿债约束扩张空间。三者嵌套，构成盈利分析的中期逻辑闭环。

🚀 贴近投资的策略研究新路径
为了让策略研究更贴近实战，我们构建了进阶四步法：

🔗 融汇 DDM 因素
打破单一因子分析，综合分子端（盈利）与分母端（利率、风险偏好）进行联动推演。

🔬 构建微观结构指标
引入量价、资金流、拥挤度等高频微观数据，辅助宏观判断。

💡 突破行业划分（反事实推理）
跳出传统的申万或中信行业分类，基于产业链逻辑进行反事实情景推演。

🎯 聚焦产业趋势
所有策略逻辑的落脚点，最终必须回到具体的产业趋势研究与行业比较之上。
